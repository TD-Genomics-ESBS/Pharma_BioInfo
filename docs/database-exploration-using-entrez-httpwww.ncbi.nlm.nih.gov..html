2.1.3 Sequence retrieval with an access number: nucleic sequence NC_045512

Return on NCBI website.

  • Select Nucleotide on the NCBI website and search for the sequence with the access number NC_045512. What is it about? How long is the genome?

  • Look at the field Comment. To which bank does the sequence NC_045512 belong? Is it a reliable entry? What sequencing technology was used? What can you say about this entry compared to the original GenBank entry (1st SARS-Cov-2 genome sequence)?

  • Look for the coding region corresponding to the N protein (nucleocapsid phosphoprotein). What do you think about the composition of this protein? What problem might such a sequence pose in a similarity search?

  • Display the genome sequence in fasta format. What are the characteristics of this format compared to the RefSeq format?
